# Novel-Shelf-Agent 📚
网文统计管理
一款专为重度网文读者打造的“多平台小说截图聚合、信息提取与阅读统计”智能体。本项目完全基于 **Claude Code** 社区的 Skill 机制与 **DeepSeek API** 敏捷开发完成。

## ⚙️ 项目背景与痛点
作为重度网文读者，追书跨越起点、番茄、晋江等多平台，各平台书架极度分散。为了统一管理，本项目实现了“截图即入库”的核心交互。
由于核心推理使用的 DeepSeek-V3/R1 缺乏原生多模态视觉能力，系统目前在 Claude Code 中挂载了一个自定义的 `image_ocr_skill`（基于开源 OCR 工具）。

在实际运行中，该方案暴露了严重的技术瓶颈：
1. **长链条推理崩塌：** 由于上游 OCR 提取的数据经常错漏，导致下游的 DeepSeek 联网检索和标签补全频繁出错，极大地消耗了无谓的 Token。

## 🛠️ 项目结构
- `app.py`: 系统主入口，串联整体业务流。
- `skills/image_ocr_skill.py`: 供 Claude Code 调用的自定义图像文本转换 Skill。
- `agents/reasoning_agent.py`: 负责接收文本、联网搜索并生成结构化表格数据的推理智能体。
